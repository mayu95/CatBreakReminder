# 🐱 CatBreakReminder · 专注养猫小游戏

一个 macOS 菜单栏小工具：把"专注用电脑"变成养成游戏——

> 专注 → 攒小鱼干 → 喂猫 → 猫长经验、升级；每 20 分钟有只小猫走来玩球，连续太久不歇则整屏柔和压暗、小猫来提醒你休息。

纯 Swift（两个源文件），用 `swiftc` 编译，**不需要 Xcode**。

## 玩法

| 机制 | 规则 |
|------|------|
| 🐟 赚小鱼干 | **只要屏幕亮着且在用**（主屏/副屏都算，开会也算工作），每累计 **20 分钟** = 1 条小鱼干 |
| 😻 喂猫长经验 | **经验只来自喂食**：心情 +25，经验 +（等级越低给得越多，前期升级快） |
| ⬆️ 升级 | 幼猫 → 奶猫 → 小猫 → 少年猫 → 成年猫 → 猫绅士 → 猫老大 → 猫王 |
| 🍽️ 会饿 | 心情随时间缓慢下降，记得回来喂 |
| 🐈 20 分钟提醒 | 每专注满 **20 分钟**（同时赚到 1 条小鱼干）：小猫走来玩会儿毛线团、停 3 秒，**不压暗** |
| 😴 45 分钟休息 | 连续使用 ≥ **45 分钟**：**整屏柔和压暗** + 小猫玩球、停 5 秒，提醒你歇歇；离开歇 ≥ 5 分钟后心情回升 |
| 🤫 不打扰 | **麦克风/摄像头在用、开会、PPT 全屏演示**时，两种提醒动画都不弹（避免打断、避免屏幕共享时被别人看到）；但这段时间**照常计入工作、小鱼干照常入账** |

### 计分判定
- **算"在用"**：空闲 < 5 分钟，或麦克风/摄像头在用（开会时不敲键盘也算）；屏幕休眠不计。
- **算"开会/演示"（仅决定是否打扰）**：麦克风占用 ∥ 摄像头占用 ∥ 有窗口全屏。
- 经验只在喂食时产生：一条小鱼干 = `max(8, 32 - (等级-1)×4)` 点经验。

## 界面与动画

- **菜单栏图标 / 弹窗头像**：用 Core Graphics 手绘的猫脸，按等级换毛色、加配饰（Lv6+ 领结、Lv7+ 皇冠、Lv8 墨镜），表情随心情变。
- **弹窗面板**（左键点菜单栏猫）：心情条、等级/经验条、**🐟 小鱼干库存**、距下一条进度、今日专注、累计完成多少个 20 分钟、喂食按钮。
- **20 分钟提醒 / 喂食 / 测试**：小猫从屏幕一侧走到中央、玩一会儿毛线团、再走掉（**不压暗**，停 3 秒），**主屏副屏同时出现**。
- **45 分钟休息提醒**：整屏轻微压暗 + 同一只玩球的小猫（气泡"该休息一下啦·已专注 X 分钟"，停 5 秒），点击穿透不挡操作，结束后压暗淡出。

## 自定义猫图（可选）

`assets/` 下放透明背景 PNG，编译时会打包进 app：
- 单张：`cat.png`
- 走路动画：`cat.png`、`cat_1.png`、`cat_2.png` …（会自动按猫的轮廓对齐到同高同基线，所以各帧原图大小不同也不会跳）

要求：**透明背景**；猫朝哪边都行（程序会统一镜像朝右）。没有图时用代码画的像素猫兜底。

## 构建 & 运行

```bash
./build.sh           # 生成 CatBreak.app（含 assets 图片）
open CatBreak.app     # 启动，菜单栏出现小猫
```
前台调试：`./CatBreak.app/Contents/MacOS/CatBreakReminder`

右键菜单栏小猫可「测试一下小猫」动画 / 退出。开机自启可在「系统设置 → 登录项」添加 CatBreak.app。

## 可调参数（`main.swift` 顶部）

| 变量 | 默认 | 含义 |
|------|------|------|
| `snackInterval` | 20 分钟 | 攒一条小鱼干所需使用时长 |
| `overworkThreshold` | 45 分钟 | 连续使用多久开始提醒休息 |
| `idleReset` | 5 分钟 | 空闲多久算"休息"并恢复 |
| `protestGap` | 10 分钟 | 两次休息提醒最小间隔 |
| `tick` | 60 秒 | 轮询间隔 |

走路猫大小、中央停留时长在 `CatOverlay.present(...)` 里（`catH`、`dwell`）。

## 数据存储

进度（小鱼干 / 经验 / 等级 / 心情 / 今日专注 / 累计专注块数）存在 `UserDefaults`（域 `com.catbreak.app`）。

## 文件

- `main.swift` —— 系统状态探测 / 游戏逻辑 / 面板 / 走路猫与休息卡片浮层 / 菜单栏
- `CatArtist.swift` —— 手绘各等级各心情的猫头像、像素猫兜底
- `build.sh` —— `swiftc` 编译并打包成 `.app`
- `assets/` —— 自定义猫图片

## 已知取舍

- "开会/演示"靠麦克风/摄像头占用 + 全屏窗口判断：极少数常驻占用麦克风的应用可能让提醒一直被压制；非全屏的小窗演示不算。
- 检测/计时按 60 秒轮询，分钟级粒度，省 CPU。
