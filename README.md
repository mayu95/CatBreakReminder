# 🐱 CatBreakReminder · 专注养猫小游戏

> 一只住在菜单栏里的猫，靠你"好好工作 + 按时休息"来养。
> 每天早上它是一只刚睡醒的奶猫；你越专注，它越进化——奶猫、小猫、机灵猫……一路到头戴皇冠、身披斗篷的 **猫王 👑**。
> 但光埋头猛干不行：不按时歇会儿，它会抗议、还会"退化"。第二天一觉醒来，重新养一只。

一个 macOS 菜单栏小工具，把"专注用电脑 + 规律休息"做成每日养成游戏。纯 Swift（两个源文件），`swiftc` 编译，**不需要 Xcode**。

## 玩法一句话

**专注 → 猫进化（=今天的评级）；按时休息 → 能进化到顶；喂小鱼干 → 猫开心。每天清零重养。**

## 评级 = 猫的种类（10 档，看今日专注时长）

| 今日专注 | 猫 | 配饰 |
|---|---|---|
| 0–30m | 刚睡醒猫 | |
| 30–60m | 幼猫 | |
| 60–100m | 奶猫 | |
| 100–150m | 小猫 | |
| 150–200m | 机灵猫 | |
| 200–260m | 学徒猫 | 🧣 围巾 |
| 260–320m | 专注猫 | 🎀 领结 |
| 320–360m | 大师猫 | 🕶️ 墨镜 |
| 360–480m（**6h，优秀**） | 猫老大 | 👑 皇冠 |
| **≥480m（8h，满分）** | **猫王** | 👑 皇冠 + 🦸 披风 |

进化越高，毛色越炫、配饰越拉风。

> **标准为什么定在 6h/8h？** 这里统计的是"屏幕在用的专注时间"（比纯深度工作宽松）。研究上"真·深度专注"每天约 4 小时就到顶，但按屏幕使用来算，一天 **6 小时**专注已经很优秀（猫老大），**8 小时**是满分（猫王）。

## 按时休息会影响评级 😴

- 想进化到顶（**猫王**）：今日专注 ≥8h **且** "连续干超 45 分钟没歇"**≤3 次**。光猛干不歇，封顶在猫老大。
- 超时 **≥5 次**：当前种类**降一档**（提醒你：会休息才是真高手）。
- 连续用满 **45 分钟**：触发休息提醒（见下）。离开歇 ≥5 分钟，状态恢复。

## 小鱼干 & 心情 🐟

- 专注每满 **20 分钟** = 1 条小鱼干。
- 喂小鱼干 → 猫**心情**变好（表情：超开心 / 开心 / 一般 / 饿了）。
- 心情和"种类/评级"是两条独立的线：种类看你专注多久，心情看你喂没喂。

## 两种提醒动画

- **20 分钟提醒**：赚到小鱼干时，小猫从屏幕一侧走到中央、玩会儿毛线团再走掉（停 3 秒，不压暗）。
- **45 分钟休息提醒**：整屏柔和压暗 + 小猫玩球，气泡「该休息一下啦」（停 5 秒，点击穿透不挡操作）。
- **喂食动画**：小猫走来玩**小鱼干**。
- 三种都**主屏副屏同时出现**，且 **开会 / 摄像头 / 麦克风 / PPT 全屏演示时统统不弹**（避免打断、避免屏幕共享被同事看到猫）——但这段时间**照常计入工作、小鱼干照常入账**。

## 计分判定（避免误计）

- 算"在用"：有键鼠输入（空闲 < 5 分钟），或正在开会/演示（摄像头/麦克风在用，不敲键盘也算）。
- **黑屏 / 锁屏 / 屏幕休眠 → 不计时**；系统睡眠唤醒后，那段时间也不会被补算。
- 主屏副屏都算，用什么 App 都算。

## 界面

- **菜单栏图标**：当前种类的猫（随专注进化、随心情变表情），跟随你正在用的那块屏幕显示（macOS 单菜单栏行为）。
- **左键点它**弹出面板：种类 + 第几阶、距下一种还需多少分钟、心情、🐟 库存、距下条进度、今日专注时长、休息情况、喂食按钮。
- **右键**：测试一下小猫 / 退出。

## 自定义猫图（可选）

提醒动画里的猫用 `assets/` 下的透明 PNG（编译时打包并自动压缩）：
- 单张：`cat.png`
- 走路动画：`cat.png`、`cat_1.png`… 多帧（会自动按猫的轮廓对齐到同高同基线，原图大小不同也不跳）

要求透明背景；朝哪边都行（程序统一镜像朝右）。没有图时用代码画的像素猫兜底。
菜单栏/面板那只是 Core Graphics 手绘的，不受 assets 影响。

## 构建 & 运行

```bash
./build.sh           # 生成 CatBreak.app（约 300KB），版本号取自 VERSION 文件
open CatBreak.app     # 启动，菜单栏出现小猫
```
前台调试：`./CatBreak.app/Contents/MacOS/CatBreakReminder`
开机自启：「系统设置 → 登录项」加 CatBreak.app。

## 版本更新

app 启动 + 每 12 小时会拉一次仓库根的 `VERSION` 文件，比当前版本新就在**右键菜单顶部**提示「🆕 有新版」（点击打开仓库），离线则静默跳过。更新方式：`git pull && ./build.sh`。

发版时把 `VERSION` 文件和 `main.swift` 里的 `appVersion` 一起改成新号再推即可。

## 可调参数（`main.swift` 顶部）

| 变量 | 默认 | 含义 |
|------|------|------|
| `snackInterval` | 20 分钟 | 攒一条小鱼干所需专注时长 |
| `overworkThreshold` | 45 分钟 | 连续使用多久触发休息提醒 / 计一次"超时" |
| `idleReset` | 5 分钟 | 空闲多久算"休息" |
| `protestGap` | 10 分钟 | 两次休息提醒最小间隔 |
| `tick` | 60 秒 | 轮询间隔（CPU 几乎为 0） |

种类档位（时长阈值/名字）在 `GameModel.tierMinutes` / `tierTitles`。

## 占用

包 ~300KB、内存 ~50MB（基本是 AppKit 框架基线）、CPU 常年 ~0%。进度存 `UserDefaults`（域 `com.catbreak.app`），每天自动归零。

## 文件

- `main.swift` —— 系统状态探测 / 每日养成与评级 / 面板 / 提醒动画 / 菜单栏
- `CatArtist.swift` —— 手绘 10 档猫头像 + 毛线团 / 小鱼干 / 像素猫兜底
- `build.sh` —— `swiftc` 编译打包成 `.app`
- `assets/` —— 自定义猫图片
